Project ini buat untuk memenuhi syarat kelulusan dari kelas Dicoding.

### Get Started
Clone repository
```bash
cd MakeFood
```

Install Modules
```bash
npm install
```

Run Development
```bash
npm run start-dev
```

Production
```bash
npm run build
```
